Product Support

Welcome to SniFit! Here you'll find help and answers to common questions.

Features

SniFit is a screenshot tool designed for macOS that helps you capture screen areas with specific aspect ratios.

Key Features

  • Aspect Ratio Screenshots - Support for 16:9, 4:3, 1:1 and other common ratios
  • Custom Ratios - Set any custom aspect ratio
  • Hotkey Support - Global hotkeys for quick screenshot capture
  • Annotations - Add text, arrows, mosaic and other annotations after capture
  • Scrolling Capture - Capture long scrolling pages
  • Text Extraction - OCR to recognize text in screenshots

FAQ

Q: How do I grant screen recording permission?

On first use, the system will prompt for screen recording permission. Please go to:

System Settings â†’ Privacy & Security â†’ Screen Recording, then enable SniFit.

Q: How do I change the screenshot hotkey?

Click the menu bar icon, select "Preferences" or "Hotkey Settings" to customize your global hotkeys.

Q: Where are screenshots saved?

By default, screenshots are copied to the clipboard. You can also configure automatic saving to a specific folder in settings.

Q: Which macOS versions are supported?

SniFit supports macOS 11.5 (Big Sur) and later.

Keyboard Shortcuts

  • Esc - Cancel screenshot
  • Enter - Confirm screenshot
  • Space - Switch aspect ratio
